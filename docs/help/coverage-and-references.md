# Coverage Map and Source References

Coverage tools show how course evidence flows into objectives and questions. They support review and explanation; they do not automatically certify factual accuracy or instructional quality.

## Coverage Map

Open the learning object's Coverage Map tab to inspect relationships among assigned materials, learning objectives, evidence, and generated questions. Use it after objectives are created and again after question generation.

Look for objectives with no evidence, objectives with no questions, heavily reused source passages, and clusters of questions that test the same subpoint. A sparse map can indicate missing coverage; an extremely dense area can indicate duplication.

## Learning-objective references

Objective references point to excerpts used during objective generation or enrichment. Open a reference to review its material, page or section, and surrounding text. If the excerpt does not support the objective, edit the objective, assign better evidence, or regenerate with clearer constraints.

## Question evidence graph

In Review & Edit, open the evidence graph for one question. The graph connects the question to its learning objective, relevant subpoint, source material, and resolved evidence. Select an evidence node to inspect the source preview.

The evidence graph is question-specific, while Coverage Map summarizes the learning object. Use the graph for close review and the map for balance across the whole design.

## PDF page highlighting

When a PDF reference has a valid page number, CREATE opens the owned source file near that page and highlights the best available excerpt. Extracted PDF text and rendered page text can differ in whitespace or line breaks, so the highlight may cover a nearby passage rather than an exact character range.

DOCX, URL, and pasted-text materials do not always have a stable printed page. Their preview uses the available extracted section or chunk instead.

## Help citations are different

Source cards in CREATE Guide cite this product manual. Selecting one opens the Help Center and highlights the cited manual section. Learning-objective and question references cite course materials and open the source-reference preview. The two reference systems intentionally use different destinations.

## Resolve weak or missing evidence

1. Confirm the source finished processing and is assigned to the learning object.
2. Preview the extracted content and locate the expected passage.
3. Remove duplicate or irrelevant sources.
4. Enrich or regenerate the objective, or regenerate the question with a clearer focus.
5. Re-open the reference and verify the new evidence before export.
